# Toon-Tank-With-UE
This game is about learning mechanism like movement, controller, sound, effect etc.

Scenerio
<p>
You enter the game.You will wait 3 seconds. When the game begin, there will be 4 target (tarret). You should destroy them and also they can kill you either. When you arrive at their shooting range, they will shoot. Be careful about that. If you finished or they killed you, the game will be restarting then. 
<p style = "font-weight: bold;">Movement</p>
<p>
Forward - W
<p>
Backward - S
<p>
Turn Left - A
<p>
Turn Right - D
<p>
Shoot - Mouse Left Click
